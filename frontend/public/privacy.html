Privacy Policy

Last updated: June 13, 2026

Wise ("we", "our", "app") is a bill-splitting application. This privacy policy explains how we handle your data.

Data We Collect

  • Split data — Restaurant names, items, prices, and participant names you enter. Stored in our database.
  • UPI IDs — Only if you voluntarily enter them for payment links. Never shared with third parties.
  • Receipt images — Temporarily processed for AI parsing. Not stored permanently after parsing.
  • Owner tokens — Stored locally in your browser (localStorage). Expires after 7 days.

How We Use Your Data

  • To calculate bill splits and generate payment links
  • To enable real-time collaboration via WebSocket
  • To provide offline access via browser cache (IndexedDB)

Data We Do NOT Collect

  • No user accounts or personal identification
  • No payment processing (we generate UPI deeplinks, but never handle money)
  • No analytics or tracking (except optional error tracking via Sentry)
  • No location data
  • No advertising or marketing data

AI Receipt Parsing

Receipt images are sent to Google Gemini Vision API for parsing. Images are processed ephemerally and not stored by Google beyond standard API logging. If you prefer not to use AI parsing, you can add items manually.

Data Retention

Split data is retained indefinitely unless you delete it. Owner tokens expire after 7 days. We may periodically clean up inactive splits.

Open Source

Wise is open source software. You can review our code at github.com/theNeuralHorizon/wise.

Changes

We may update this policy. Continued use of the app constitutes acceptance of changes.
